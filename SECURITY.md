# Security & Privacy Policy / 安全与隐私政策

## 🛡️ Core: Local Inference Architecture / 本地推理架构
* **SC**: 本项目采用本地推理架构。所有 OCR 识别与模型决策均在您的本地设备完成。
* **TC**: 本項目採用本地推理架構。所有 OCR 識別與模型決策均在您的本地設備完成。
* **JP**: 本プロジェクトはローカル推論アーキテクチャを採用しています。すべてのOCR識別とモデルの意思決定は、お客様のローカルデバイス上で行われます。
* **EN**: This project utilizes a local inference architecture. All OCR recognition and model decision-making are performed locally on your device.

## 🔒 Zero Data Transmission / 零数据传输
* **SC**: 您的对局数据和账号信息绝不上传。我们不设任何后端服务器来存储用户隐私。
* **TC**: 您的對局數據和賬號信息絕不上傳。我們不設任何後端服務器來存儲用戶隱私。
* **JP**: 対局データやアカウント情報は一切アップロードされません。ユーザーのプライバシーを保存するためのバックエンドサーバーは一切存在しません。
* **EN**: Your game data and account information are never uploaded. We do not maintain any backend servers to store user privacy.

## 🚫 Non-Invasive Safety / 非侵入式合规性
* **SC**: 严禁读取游戏内存或修改本地文件。通过屏幕识别技术从物理层面确保合规。
* **TC**: 嚴禁讀取遊戲內存或修改本地文件。通過屏幕識別技術從物理層面確保合規。
* **JP**: ゲームメモリの読み取りやローカルファイルの改ざんは一切行いません。スクリーン識別技術により、物理的なレベルでの安全性を確保しています。
* **EN**: Memory reading or modification of local game files is strictly prohibited. We ensure safety through screen recognition technology.
