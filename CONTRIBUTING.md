## 🛠️ How to Contribute / 如何贡献 / 如何貢獻 / 貢献方法

### 1. Submit Logs / 提交牌谱 / 提交牌譜 / 牌譜の提供
* **EN**: Share high-quality game data via [Issue Templates](https://github.com/halfofsever/MahjongMaster_AI/issues).
* **SC**: 请通过[问题模板](https://github.com/halfofsever/MahjongMaster_AI/issues)分享高质量牌谱数据。
* **TC**: 請通過[問題模板](https://github.com/halfofsever/MahjongMaster_AI/issues)分享高品質牌譜數據。
* **JP**: [問題テンプレート]を使用して、高品質な牌譜データを共有してください。

### 2. Strategy Discussions / 策略讨论 / 策略討論 / 戦術ディスカッション
* **EN**: Participate in Discord or GitHub Issues for model strategy optimization.
* **SC**: 在 Discord 或 GitHub Issues 中参与模型策略优化讨论。
* **TC**: 在 Discord 或 GitHub Issues 中參與模型策略優化討論。
* **JP**: DiscordまたはGitHub Issuesで、モデルの戦術最適化に関する議論に参加してください。

### 3. Bug Reports / 报告问题 / 報告問題 / バグ報告
* **EN**: Help us refine OCR accuracy or report software glitches.
* **SC**: 帮助我们提升 OCR 识别准确度或报告软件错误。
* **TC**: 幫助我們提升 OCR 識別準確度或報告軟件錯誤。
* **JP**: OCRの認識精度向上への協力や、ソフトウェアの不具合を報告してください。
